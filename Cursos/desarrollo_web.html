Mi Primera Aplicación Práctica (HTML y CSS)

En esta etapa del curso, he puesto en práctica las bases de HTML y CSS para mejorar el diseño de todo mi portafolio. He logrado:

  • Separar el contenido del estilo: Utilizando un archivo CSS externo para gestionar la apariencia de todas las páginas, lo cual es una buena práctica en el desarrollo web.
  • Dar estilo a los elementos: Aplicando colores, fuentes y espaciados para hacer el sitio más estético y fácil de leer.
  • Mejorar la presentación: He estilizado tablas y enlaces para que se integren con el diseño general del sitio.
