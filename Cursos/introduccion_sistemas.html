Puntos Clave Aprendidos:

  • Componentes de hardware y software de un sistema de cómputo.
  • Funcionamiento básico de los sistemas operativos.
  • Conceptos de redes de computadoras (muy básicos).
  • Principios de programación y algoritmos.
  • Aplicaciones de la computación en diferentes campos.

Componentes Fundamentales de un Sistema de Cómputo

Componente Tipo Función Principal
CPU (Procesador) Hardware Ejecuta instrucciones y procesa datos.
RAM (Memoria RAM) Hardware Almacena datos temporalmente para acceso rápido.
Disco Duro/SSD Hardware Almacena datos de forma permanente.
Sistema Operativo Software Gestiona los recursos del hardware y software.
Aplicaciones Software Programas diseñados para tareas específicas.
