Puntos Clave Aprendidos:

  • Diseño y modelado de bases de datos relacionales (Diagramas ER).
  • Lenguaje SQL: Consultas, inserción, actualización y eliminación de datos.
  • Normalización de bases de datos para optimizar la estructura.
  • Conceptos de integridad referencial y transacciones.
  • Uso de sistemas gestores de bases de datos (SGBD) como [ej. MySQL, PostgreSQL].

Conceptos Esenciales de Bases de Datos

Concepto Descripción Breve
Entidad Representación de un objeto o cosa del mundo real (ej. Persona, Producto).
Atributo Propiedad que describe una entidad (ej. Nombre, Edad).
Relación Asociación entre dos o más entidades (ej. Un Cliente "realiza" un Pedido).
Clave Primaria (PK) Identificador único para cada registro en una tabla.
Clave Foránea (FK) Campo que establece un vínculo entre dos tablas.
